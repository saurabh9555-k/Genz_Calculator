✨ Overview - Full Fun
The Ultra-Glass Vibe Calculator is not just a tool—it's an experience. Designed for the modern user, it combines a powerful calculator with a stunning glassmorphism interface. Choose from preset themes or upload your own images, GIFs, or videos as the background. Adjust opacity and blur in real time to match your vibe.

Perfect for students, creators, and anyone who appreciates beautiful design.

🧮 Features
All Essential Operations
Addition (+), subtraction (-), multiplication (*), division (/), square (x²), square root (√), power (x^y), percentage (%), sign toggle (±), backspace, clear.

Memory & History

Memory buttons: M+, MR, MC, MS

Live history bar showing last 5 calculations

Clear history with one tap

Live Theme Studio

6 preset themes: Glass, Dark, Light, Neon, Sunset, Ocean

Upload your own images, GIFs, or MP4 videos as background

Real‑time opacity (0–0.8) and blur (0–50px) sliders

Instant preview – no reload needed

Glassmorphism 2.0 Design

Ultra‑transparent panel (rgba(255,255,255,0.15) default)

Strong blur (30px) and saturate effect

Soft shadows, subtle borders, and smooth animations

Buttons scale on press for tactile feedback

Responsive & Mobile‑First

Works flawlessly on phones, tablets, and desktops

Touch‑optimized with no 300ms delay

🚀 Getting Started
Installation
Download the index.html file from the latest release.

Open it in any modern browser (Chrome, Firefox, Safari, Edge).

Start calculating! Click the 🎨 button to customize your vibe.

No build tools, no dependencies—just one file.

Usage Tips
Preset Themes: Click any theme name in the panel to apply instantly.

Custom Background: Click "Image/GIF" or "Video" to upload your own media.

Adjust Glass: Use the sliders to change transparency and blur.

Memory: Store numbers with M+, recall with MR, clear with MC, and store directly with MS.

History: Click the 🗑️ button to clear history.


🛠️ Built With
HTML5 – structure

CSS3 – styling, animations, glass effects

Vanilla JavaScript – calculator logic, theme engine, media handling

No frameworks, no libraries – pure web standards

🎨 Customization
You can easily tweak the calculator to your liking:

Colors: Modify the CSS variables in the <style> block.

Button Layout: Adjust the grid in the HTML.

Default Theme: Change the presets object in the JavaScript.
